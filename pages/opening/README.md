# Opening

사이트 첫 진입 시 Studio Midnight의 로고와 짧은 파형 연출을 보여주는 오프닝 페이지입니다.

## 파일 구성

- `index.html`: 오프닝 화면 진입점입니다.
- `page.css`: 레이아웃, 로고, 파형 애니메이션을 관리합니다.
- `opening.js`: 자동 홈 전환과 `입장하기` 버튼을 처리합니다.

## 동작 방식

- 첫 방문에서는 약 6초 뒤 `pages/home/`으로 이동합니다.
- `입장하기`를 누르면 즉시 홈으로 이동합니다.
- 같은 탭에서 다시 루트에 접속하면 `sessionStorage`를 사용해 오프닝을 건너뜁니다.
- `pages/opening/?preview=1` 주소에서는 자동 이동 없이 오프닝을 확인할 수 있습니다.
