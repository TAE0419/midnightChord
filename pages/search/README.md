# Search

Studio Midnight의 통합 검색 화면을 담당하는 폴더입니다.

## 파일 구성

- `index.html`: `common.html`을 불러온 뒤 검색 콘텐츠 슬롯과 공용 재생바를 연결합니다.
- `search.js`: 검색어와 결과 유형 필터를 처리하고 검색 화면을 렌더링합니다.
- `page.css`: 검색 결과 목록, 필터, 빈 결과 화면의 전용 스타일을 관리합니다.

## 검색 범위

검색어는 곡, 앨범, 아티스트, 플레이리스트, 팟캐스트 시리즈와 에피소드 정보를 대상으로 비교합니다. 필터는 전체, 곡, 앨범, 아티스트, 플레이리스트, 팟캐스트로 나뉩니다.

곡 결과를 선택하면 기존 공통 하단 재생바로 재생이 연결됩니다. 검색 화면은 최신 아티스트 데이터를 함께 로드하지만, 통합 검색 렌더러는 `search.js`가 유지합니다.

## 작업 확인

1. 검색어 입력 시 결과 수와 목록이 갱신되는지 확인합니다.
2. 각 필터를 선택했을 때 해당 종류의 결과만 표시되는지 확인합니다.
3. 결과가 없을 때 빈 결과 화면이 보이는지 확인합니다.
4. 곡 행을 선택했을 때 하단 재생바 정보가 바뀌는지 확인합니다.
