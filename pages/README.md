# Pages

각 페이지는 자기 폴더 안의 `index.html`과 `page.css`를 중심으로 작업합니다.

공용 레이아웃, 메뉴, 하단 재생바, 색상, 폰트는 루트의 `css/common.css`와 `js/trackit-app.js`를 공유합니다.

- `home/`: 홈
- `browse/`: 둘러보기
- `album/`: 앨범 상세
- `artists/`: 아티스트 목록
- `artist-detail/`: 아티스트 상세
- `podcasts/`: 팟캐스트
- `playlist/`: 플레이리스트
- `search/`: 검색
- `mypage/`: 마이페이지
- `settings/`: 설정
